ICON FIX v2

Upload these files to the ROOT of your GitHub repository:
- icon-192.png
- icon-512.png
- manifest.json
- service-worker.js

Important: they must be beside index.html, not inside a folder.

Then:
1. Wait 2-5 minutes for GitHub Pages to update.
2. On your Samsung, delete the old home screen shortcut.
3. Open Chrome.
4. Go to your app link.
5. Add this to the end of the URL once:
   ?v=2
   Example:
   https://YOURNAME.github.io/chris-inspection-assistant/?v=2
6. Open that.
7. Chrome menu ⋮ > Add to Home screen or Install app.

If the old icon still appears:
- Chrome > Settings > Site settings > All sites
- Find github.io or your app site
- Clear & reset
- Then open the app link again and add it again.
